## Was sie machen
Cordium is a free and open source, self-hosted, identity-based sandbox platform built on Kubernetes and the Octelium framework. It provides isolated, reproducible general-purpose sandboxes for developers, AI agents, and automated workloads accessible through web terminals, SSH, CLI, and gRPC APIs. Unlike traditional sandbox solutions, Cordium assigns each sandbox a dedicated Octelium identity so that databases, SSH servers, HTTP APIs, Kubernetes clusters, and internal services are accessed through an identity-aware, secretless access layer — meaning API tokens, passwords, SSH private keys, and other long-lived credentials never need to be placed inside the sandbox. Key capabilities include declarative YAML-based workspace configuration with pre-built templates using Kubernetes VolumeSnapshots for fast cold-start times, L7-aware attribute-based access control (ABAC) with policy-as-code via CEL and OPA, OIDC/SAML 2.0/GitHub OAuth2/FIDO2/WebAuthn/TOTP authentication, OpenTelemetry-native auditing and SIEM integration, and full self-hosting under the Apache-2.0 license with no proprietary control plane or vendor lock-in.
