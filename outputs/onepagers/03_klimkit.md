## Was sie machen
Klimkit is an open-source tool that keeps an agent-ready machine reproducible. One repository owns the local instructions, harness packs, services, dashboards, and machine-specific settings needed to make a fresh VM behave like a consistent working environment. Users edit the repo, preview exactly what will change, apply it locally, and use normal Git flow to carry the same operator setup to another machine. Klimkit is designed for long agent runs (e.g. 7.5-hour Codex sessions) where inspectable artifacts — task notes, checks, diffs, screenshots, reports, reflections, and final reviews — must live in the repo. Its Switchboard feature enables parallel agent work across 5–7 Codex/code-server workspaces open across machines, each on its own branch and worktree, while keeping the harness, browser profile, services, and notifications synced. Built with Python 3.11+ and integrated with Tailscale for networking.
