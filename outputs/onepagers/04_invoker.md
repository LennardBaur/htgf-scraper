## In einem Satz
An agentic IDE for orchestrating AI coding agents in parallel with a kanban board, full IDE, and worktree support.

## Was sie machen
Invoker is a native macOS application described as "Linear + Superhuman for orchestrating agents." It enables developers to run multiple AI coding agents (Claude Code and Codex) in parallel using a kanban-style task board. Each task gets a full built-in IDE with editor, terminal, and git diff viewer, plus worktree support. The product is keyboard-first with vim-style navigation, a ⌘K command palette, and ready notifications when agent tasks are complete. Users can reference other tasks via @-mentions and search across all tasks. Invoker uses the user's existing Claude/Codex CLI credentials and billing, offering a separate subscription with monthly, annual, and lifetime plans plus a free tier with 1 hour of agent time per day after a 3-day trial. Currently at v0.0.1, requiring macOS 13+ and Apple Silicon.
