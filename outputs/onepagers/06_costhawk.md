## Was sie machen
CostHawk is an AI adoption monitoring platform for engineering teams that tracks who is using AI tools, how much they cost, and how adoption is trending — across providers including OpenAI, Anthropic, Google, xAI, Mistral, DeepSeek, AWS Bedrock, Azure OpenAI, and more. It supports three integration methods: MCP telemetry (local-first, no admin keys needed), admin API sync for organization-wide data, and wrapped proxy keys that sit between the user's code and the provider. Features include team benchmarks with private role-based access, budget alerts, cost optimization recommendations, a savings engine, data export (CSV/JSON), a REST API, and a 5-minute setup. CostHawk was built by founder Chase Dillingham out of his experience at TrainMyAgent, where unexpected AI API bills with no per-model or per-project breakdown motivated the creation of the tool. The product is offered as Individual ($19/mo), Team ($99/mo), and Enterprise (custom) plans, all with a 30-day free trial. The company operates as Outperform Ventures LLC (dba TrainMyAgent).
