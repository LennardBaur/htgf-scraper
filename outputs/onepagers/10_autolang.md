## In einem Satz
An ultra-low latency, strictly-typed scripting language designed for AI Agents to write code safely within a zero-trust, sandboxed execution environment.

## Was sie machen
Autolang is a scripting language built specifically for AI Agents to write and execute code safely, quickly, and at low cost. Rather than replacing Python or JavaScript, Autolang acts as an orchestration layer (a "zero-trust membrane") that wraps existing functions from any ecosystem—npm, Python, C++—and exposes only explicitly registered bindings to the AI. A static compiler validates all AI-generated code before execution, enforcing strict typing, opcode execution limits (to prevent infinite loops), and fine-grained null-safety controls. The language uses Kotlin-inspired syntax with TypeScript-like data structures, optimizing for total execution time (compile + runtime) rather than raw throughput. Internal benchmarks show ~900 lines executed in ~25ms with ~5.1MB RAM per compiler instance. Autolang is MIT-licensed and sponsored by ADA GROUP.
