## In einem Satz
A cross-provider memory layer for LLM apps that reduces token costs by up to 80% through intelligent conversation signal storage and retrieval.

## Was sie machen
Street AI is a continuously learning memory layer that sits between an application and any LLM API. It stores conversation turns as "signals" organised into "stacks", automatically decays stale data, and retrieves only contextually relevant information on each turn — enabling the app to send a tiny, focused prompt instead of the full conversation history. In a 16-turn benchmark, input tokens dropped by 55–80% per turn (average 68%), with savings growing as conversations lengthen. The library is available as the open-source Python package `streetai-memory` on PyPI and ships with drop-in adapters for OpenAI, Anthropic, Google Gemini, DeepSeek, Together, Groq, and any OpenAI-compatible endpoint. It uses a local SQLite file for persistent, per-user or per-session memory that survives restarts. The project is currently in Alpha (v0.2.0).
