## Was sie machen
Kilo Code is an open-source AI coding agent that integrates with VS Code and JetBrains IDEs to help developers plan, build, ship, and iterate on code faster. Key features include parallel agents, a diff reviewer, multi-model comparisons, and automatic AI-powered code reviews triggered when a pull request is opened. The product supports multiple AI models and has processed over 30 trillion tokens, with a community of more than 3 million users ("Kilo Coders"). It has received multiple top-ranked Product Hunt launch awards, including #1 of the week and #1 of the day.
