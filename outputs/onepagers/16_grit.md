## Was sie machen
Grit is an open-source project that reimplements Git entirely in Rust using an agent-driven development approach. It consists of two main components: grit-lib, which provides Git internals as typed, reentrant Rust modules (covering objects, packs, index, refs, revisions, diff, merge, config, ignore rules, hooks, and more), and grit-cli, a Git-compatible command-line interface built to pass Git's own upstream test suite. Inspired by Anthropic's experiment building a C compiler via agents and test suites, the project aims to deliver the long-desired dream of a full, library-grade Git implementation that does not rely on fork-exec wrappers. As of the latest harness run, 21,284 of 41,220 in-scope tests pass (~51.6% overall), with 140+ Git commands implemented. The project is authored by Scott (GitHub: schacon) and is available on crates.io and via a shell installer.
