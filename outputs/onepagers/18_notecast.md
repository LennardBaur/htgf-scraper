## In einem Satz
A local LLM-powered note engine that automatically summarizes, embeds, and organizes notes into an evolving knowledge graph.

## Was sie machen
NoteCast is a local engine that enriches raw notes and organizes them into an evolving knowledge graph. Each note is automatically summarized, keyword-extracted, and embedded before it reaches the organization stage. A three-stage LLM pipeline then assigns notes to themes, splits those themes into subtopics as they grow, and rewires connections across the graph as patterns emerge. The pipeline consists of four stages: NoteProcessor (summarization, topic extraction, embedding), Classify (vector-similarity grouping and theme assignment), Organize (sub-cluster detection and theme splitting), and Consolidate (macro-level restructuring into a multi-parent DAG). The tool supports multiple LLM providers including OpenAI, Anthropic, Gemini, DeepSeek, Codex, and Ollama, and exposes both a CLI and a REST API. It is currently an open-source, solo-developer side project hosted on GitHub, described as usable but not yet stable.
