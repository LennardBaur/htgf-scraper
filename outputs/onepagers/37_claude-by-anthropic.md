## Was sie machen
Claude is a family of foundational AI models developed by Anthropic, an AI research company focused on building reliable, interpretable, and steerable AI systems. Claude serves as an AI assistant capable of handling tasks at any scale, including coding, writing, design prototyping, memory import, and integration with tools like Microsoft Word. Recent launches include Claude Import Memory (for switching from ChatGPT), Claude Code Routines, Claude for Word, Claude Design, and Claude Desktop Buddy.
