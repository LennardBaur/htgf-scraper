## In einem Satz
A Chrome extension that extracts AI-generated HTML from ChatGPT, Claude, and Gemini and publishes it directly to a live website with one click.

## Was sie machen
HTML Deployer is a Chrome browser extension that turns AI-generated HTML into a live website quickly and efficiently. It detects and extracts HTML code blocks from supported AI chat platforms (ChatGPT, Claude, Gemini), provides a multi-device preview (desktop, tablet, mobile), and enables direct publishing to multiple hosting targets including Netlify, GitHub-based workflows, FTP servers, and self-hosted agents. Users can also export pages as ZIP packages for manual hosting. The tool maintains a deployment history, auto-suggests filenames from HTML titles, and is designed to reduce the manual steps between AI output and a publicly accessible URL. Target users include developers, founders, marketers, agencies, and beginners.
