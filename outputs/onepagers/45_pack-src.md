## In einem Satz
Recursively pack source code directories into clean ZIP archives — fast, smart, and ignore-aware.

## Was sie machen
pack-src is an open-source Node.js CLI tool and programmatic API for recursively packing source code directories into clean ZIP archives. It features smart ignoring (respects .packsrcignore, .gitignore, .dockerignore, .npmignore, and built-in defaults), streaming ZIP compression that never loads entire files into memory, deterministic and reproducible output, zip-slip-safe path validation, and a polished CLI with progress spinners and colored output. It requires zero configuration out of the box and achieves near-native performance (~180ms for a 1,000-file TypeScript project) via concurrent directory traversal with fast-glob and direct file streaming. Published as an npm package by GitHub user muhammadmuzzammil1998 under an MIT license.
